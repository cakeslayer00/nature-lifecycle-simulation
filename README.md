# nature-lifecycle-simulation

Small CLI application, showcasing lifecycle of prey and predators. Goal in mind was to develop applicaiton in OOP style, utilizing clean code principles

# requirements

- `JDK 26`
- `Maven`

# render issues

For those who run application on windows/IDE terminal, it might not render as intended. Recommend using WSL/Git Bash console for those who is on Windows. Your terminal font must also support glyphs to display emoji icons 🤓 

# how to run

1. Clone and enter directory
```
git clone https://github.com/cakeslayer00/nature-lifecycle-simulation && cd nature-lifecycle-simulation
```
2. Build and run **from a real terminal** (not the IDE console):
```
mvn clean package && java -cp target/simulation-1.0-SNAPSHOT.jar Main
```
3. Enjoy!
